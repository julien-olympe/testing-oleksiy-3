Wind Power

You are going to assemble an wind power plant status investigation app.

Login screen
User can register and login

Home screen
Lists the projects the user is assigned to
Each item can be viewed by double click. The items show the status of the project (In Progress, Finished).
There is a button: Start project, which allows creating a new project

If Clicked an existing project item:
The Ongoing Project screen is opened

Ongoing Project Screen:
Indicates the Wind Turbine powerplant name
Lists the Parts of Wind Turbines and check ups for each part in the middle. On the right is shows the docu or pics that are in existence in the database for each part.
The user can click on the parts and set status for each check ups. The check ups are taked from the databse for each part of a particular Wind Turbine.
The status can be bad, average, good
There is the button "Finish Report" in the top right corner.
If the user clicks the Finish report button, the list of check ups and parts are composed into a PDF and automatically downloaded. The projects gets the Finished status.

If the user clicks the Start project button,
The Start Project screen is opened.
The Start project Screen UI:
Selector of Powerplants (dropdown) which openes the list of parts an check ups. The user annot edit the lists only add the powerplant as a whole with all the parts and checkups.

Below is the Create button
Once the user clicks the create button, the project is created, assigned to the user and listed inthe users home screen.
