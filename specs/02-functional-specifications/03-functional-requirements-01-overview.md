# 3. Functional Requirements

## 3.1 Use Case Overview

The system supports the following primary use cases:
1. User Registration
2. User Login
3. View Projects List
4. Open Project
5. Start New Project
6. Select Powerplant
7. Create Project
8. View Ongoing Project
9. Set Checkup Status
10. View Documentation
11. Finish Report
12. Download PDF Report

## 3.2 Use Case Specifications

Detailed specifications for each use case are provided in the following files:
- Use Cases 1-6: `03-functional-requirements-02-use-cases-01.md`
- Use Cases 7-12: `03-functional-requirements-02-use-cases-02.md`
